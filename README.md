# Basic Inspect Command Program


# the code uses a Node.js command called 'inspect'

# the program looks through and displays the total number of lines, first requested number of lines, and last requested number of lines.

# the command uses wc, head, and tail Linux commands


# To run, use the format:

# node tool.js inspect (file name) (number of lines)

# ex: node tool.js inspect sample.txt 2
